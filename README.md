# Zain Bin Asad — Personal Digital Marketing Portfolio

A single-page portfolio website built with **Vite**, **React**, and **Tailwind CSS**. No backend; static frontend only.

## Assignment checklist

- **Header:** Name + navigation (Home, About, Services, Projects, Contact)
- **Hero:** Headline, value proposition, call-to-action
- **About:** Brief intro + key strengths/skills
- **Services:** 3 services with how each helps businesses
- **Projects:** 3 projects with objective and strategy
- **Contact:** Email, phone, LinkedIn (optional GitHub link)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production / deploy

```bash
npm run build
```

Output is in `dist/`. You can deploy that folder to:

- **Vercel:** `npx vercel` (or connect the repo)
- **Netlify:** Drag `dist` to Netlify or connect repo with build command `npm run build` and publish directory `dist`
- **GitHub Pages:** Use a workflow that runs `npm run build` and deploys `dist`
